## 🚀 Inicio rápido

Si es tu primera vez usando el CRM:

1. Abre **[01-Introduccion](Manual-Usuario/01-Introduccion.md)** para entender qué hace el sistema
2. Ve a **[02-Login-Autenticacion](Manual-Usuario/02-Login-Autenticacion.md)** para iniciar sesión
3. Según tu rol:
   - **Si eres una asesora:** consulta [04-Inbox-Conversaciones](Manual-Usuario/04-Inbox-Conversaciones.md) para ver tus clientes y conversaciones
   - **Si eres administrador:** comienza por [03-Clientes](Manual-Usuario/03-Clientes.md) para importar o crear la base de clientes

---

## 📱 Acceso

- **URL:** https://crm.lainternacional.com.ar
- **Compatible con:** Desktop (admin), mobile/tablet (asesoras)
- **Navegadores recomendados:** Chrome, Firefox, Safari (versiones modernas)

---

## ⚠️ Notas importantes

- **El sistema es paralelo:** funciona independientemente del sistema actual de La Internacional. Los datos se replican manualmente según necesidad.
- **WhatsApp:** requiere credenciales de Meta Cloud API que el admin debe configurar en Configuración → Credenciales.
- **Permisos:** los contenidos visibles dependen de tu rol (Admin o Asesora).
